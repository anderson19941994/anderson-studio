## 🌟 Sobre el Proyecto
Este portafolio fue construido con un enfoque en la **velocidad, el diseño limpio y la integración de IA**. El objetivo es conectar con clientes potenciales ofreciendo una presencia digital sólida.

### 🛠️ Tecnologías Utilizadas
*   **HTML5 & CSS3:** Estructura semántica y estilos modernos.
*   **JavaScript:** Interactividad y lógica de navegación.
*   **Gemini AI:** Integrada en el flujo de trabajo para optimización de soluciones.
*   **FontAwesome:** Iconografía profesional.
*   **Google Fonts:** Tipografía *Plus Jakarta Sans* para una lectura premium.

## 📱 Características
- [x] **Diseño Responsive:** Adaptado para móviles, tablets y escritorio.
- [x] **Botón de WhatsApp Flotante:** Contacto directo y rápido.
- [x] **Sección de Proyectos:** Galería visual de trabajos realizados.
- [x] **Redes Sociales:** Integración con Instagram y TikTok.
- [x] **SEO Friendly:** Optimizado para motores de búsqueda.
